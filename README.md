# Online Medicine Shop

Web Technologies Group 05 Project

## Requirements

XAMPP installed with Apache and MySQL running.

## Database Setup

Open phpMyAdmin at http://localhost/phpmyadmin

Import the file task1_23540323/database.sql

This will create the database and sample medicines. Admin and customer accounts are created automatically on first page load.

Default admin login: admin@shop.com / admin123

Default customer login: customer@shop.com / customer123

## How to Run

Copy the entire project folder into C:/xampp/htdocs/

Open your browser and go to http://localhost/online_medicine_shop/

## How Each Task Run

Task 1: http://localhost/online_medicine_shop/task1_23540323/

Task 2: http://localhost/online_medicine_shop/task2_23540353/

Task 3: http://localhost/online_medicine_shop/task3_23544053/

## Project Structure

task1_23540323 handles login, register, home page, medicine browsing and profile.

task2_23540353 handles admin dashboard, medicines, categories, customers and orders.

task3_23544053 handles cart, checkout, payment and bill.
